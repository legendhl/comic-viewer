# ComicViewer

由于Mac OS自带的图像预览软件使用体验不佳，为了在Mac下更方便的看漫画开发了本软件，开发目标是实现windows下常见漫画浏览软件的基本功能。

目前支持常见图像格式及zip文件，上一张、下一张，循环浏览。
